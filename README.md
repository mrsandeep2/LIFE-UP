# LIFE-UP

FocusEdge (LIFE-UP) is a personal life OS for habits, tasks, focus sessions, and progress tracking.

## Tech Stack
- Vite + TanStack Start
- React + TypeScript
- Tailwind CSS
- Supabase

## Getting Started

### Prerequisites
- Node.js 18+ (or Bun 1.3+)

### Install

With Bun:
```bash
bun install
```

With npm:
```bash
npm install
```

### Environment
Create a `.env` file with the following keys:
```
VITE_SUPABASE_URL=...
VITE_SUPABASE_PUBLISHABLE_KEY=...
VITE_SUPABASE_PROJECT_ID=...
```

### Run

With Bun:
```bash
bun dev
```

With npm:
```bash
npm run dev
```

## Build

With Bun:
```bash
bun run build
```

With npm:
```bash
npm run build
```

## Scripts
- `dev`: start the dev server
- `build`: production build
- `preview`: preview the production build
- `lint`: run ESLint
- `format`: run Prettier
